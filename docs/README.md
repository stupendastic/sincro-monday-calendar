# Documentación del Sistema

Esta carpeta contiene la documentación del sistema de sincronización Monday.com ↔ Google Calendar.

## Archivos

### `PLAN_PRUEBAS_MANUALES.md`
Plan detallado para realizar pruebas manuales del sistema de sincronización bidireccional.

**Contenido:**
- Instrucciones paso a paso para pruebas
- Casos de uso específicos
- Verificación de funcionalidades
- Troubleshooting de problemas comunes

## Uso

Esta documentación complementa el README principal del proyecto y proporciona guías específicas para:

1. **Testing Manual**: Cómo probar el sistema de forma manual
2. **Verificación**: Cómo verificar que todo funciona correctamente
3. **Troubleshooting**: Cómo resolver problemas comunes

## Mantenimiento

- Actualizar la documentación cuando se añadan nuevas funcionalidades
- Mantener las instrucciones sincronizadas con el código
- Añadir nuevos casos de uso según sea necesario
